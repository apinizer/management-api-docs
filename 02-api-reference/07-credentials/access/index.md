# Credential Access Management

This section contains endpoints for managing access to credentials.

## Endpoints

- [Get Granted Access List](./get-granted-access-list) - Get list of users/organizations with access to a credential
- [Grant Access](./grant-access) - Grant access to a credential to users or organizations
- [Revoke Access](./revoke-access) - Revoke access to a credential from users or organizations
