# Certificates CRUD Operations

This section contains endpoints for creating, reading, updating, and deleting certificates.

## Endpoints

- [List Certificates](./list-certificates) - Get all certificates for a project
- [Get Certificate](./get-certificate) - Get detailed information about a specific certificate
- [Get Truststore Certificates](./get-truststore-certificates) - Get certificates from truststore
- [Create Certificate](./create-certificate) - Create a new certificate
- [Update Certificate](./update-certificate) - Update an existing certificate
- [Export Certificate](./export-certificate) - Export a certificate
- [Delete Certificate](./delete-certificate) - Delete a certificate
