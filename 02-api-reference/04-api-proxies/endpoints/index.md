# API Proxy Endpoints

This section contains endpoints for managing API proxy endpoints (routes/paths).

## Endpoints

- [List Endpoints](./list-endpoints) - Get all endpoints for an API proxy
- [Get Endpoint](./get-endpoint) - Get detailed information about a specific endpoint
- [Create Endpoint](./create-endpoint) - Create a new endpoint
- [Update Endpoint](./update-endpoint) - Update an existing endpoint
- [Delete Endpoint](./delete-endpoint) - Delete an endpoint
- [Update Endpoint Status](./update-endpoint-status) - Enable or disable an endpoint
- [Update Endpoint WSA](./update-endpoint-wsa) - Update SOAP WS-Addressing settings for an endpoint
- [Update Endpoint Cache](./update-endpoint-cache) - Update cache settings for an endpoint
- [List Endpoint Policies](./list-endpoint-policies) - List policies applied to an endpoint
